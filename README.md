# URL Shortener Web Application

This is a basic URL Shortener web application built using Flask and SQLite.

## Features
- Shortens long URLs
- Redirects to original URLs
- Stores URLs in database
- Displays URL history
- Validates URLs

## Technologies Used
- Python
- Flask
- SQLite
- SQLAlchemy
- HTML, CSS

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   python app.py
3. Open browser:
   http://127.0.0.1:5000/
